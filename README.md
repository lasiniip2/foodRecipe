# Foodish – Food Blogging Platform
**Foodish** is a full-stack food blogging web application built using the **MERN stack** (MongoDB, Express.js, React, Node.js).  
It allows users to upload, explore, and share food recipes while interacting with others.

## Features
- **User Accounts & Authentication** – Sign up, log in, and manage your profile securely.
- **Post Recipes** – Upload food recipes with descriptions, images, and ingredients.
- **Favorites** – Add your favorite recipes to a personal list.
- **Engagement** – Print and share recipes shared by other users.

## Tech Stack
**Frontend:** React, CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT (JSON Web Tokens)
**Other Tools:** Git, Postman
